## my_vim说明
这个Git仓库只是用来保存我目前在用的vim配置，并非对vim或者其插件的开发。  
所以，也可以看到，这里的管理方式也是非常简单粗暴了。  
例如，插件目录bundle中其实含有不少Git仓库的，在这里我只是将它们简单压缩后，就传上来啦～  

## 使用说明
git clone本项目仓库后，进入其目录，运行：  
```
$ ./lay_out_vim_config.sh
```
原有的.vim和.vimrc会被重命名为\_vim_orig和\_vimrc_orig。
